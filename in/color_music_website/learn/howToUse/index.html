Using the App

A few tips to help get you started.

Browsers

For optimum quality, use Google Chrome. If the app does not load properly on the first try, simply reload the page.

Endless possibilities.

Many songs are available for download as midi files. You may use a search engine to find sources from which you can download the files. Simply drag the file into Color Piano, and the interface will turn red. Drop the file to play it.

Practice makes perfect!

Take a few minutes to explore the icons at the top of the app. As you learn how to use them, you will become more familiar with Color Piano.
